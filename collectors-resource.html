The Collector's Resource app

Project Overview:

A project developed in the full time Java course at We Can Code IT coding bootcamp. This project is an app for collectors of various things, whether it be stamps, cars, Pokemon cards, etc...

Features:

This app features a populated list of items in many categories that users can add to their collections and also to a wishlist. The categories are called genres and the sub categories are called categories. The user navigates to a sub category to begin browsing the database.

Users can create an account and login. User accounts have password encryption for added security.

Admin functionality allows admin users to change things about the site, like adding items to the catalogue, making new categories, changing the roles of users to admin or back to regular user, and deleting users. A suggestion page allows users to submit suggestions about adding things or changing things to the website in the future.

What We Learned:

This project was a learning experience for all who were involved. The team strengthened their skills in Java backend structure, password encryption, front end implementation of role permissions.

Personal Contributions:

I built a majority of the UI structure of the site as well as the user role permissions for certain parts of the site. I also worked in the back end, making the data models, controllers, and services for the items listed in the site.

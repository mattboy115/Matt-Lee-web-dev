Jac Enterprises

Project Overview:

A project developed in the full time Java course at We Can Code IT coding bootcamp. This project is an educational app for kids that provides a fun and colorful way to learn. It includes challenging questions about math, art, and the animal kingdom.

Features:

This app features three different sections. Math, Art, and Animals. Each section has three different difficulties. Users can choose a section and begin answering questions.

What We Learned:

I personally learned a lot about React because this was my first real React project. The rest of the team leaned about user accounts and password encryption as well as strengthening their skills in Java backend mechanics.

Mobile Screenshots:

Mobile responsiveness has been implemented so that Jac Enterprises can be seamlessly used on phones and tablets as well as computers.
